E-com Website (Tomato) is a full-stack e-commerce application built with the MERN stack (MongoDB, Express, React, Node.js).
--------------------(Still on progress)-----------------------

Key Feature:-
User Authentication (Register/Login)
Product Listing and Categories
Shopping Cart and Checkout
Order Management for Users and Admin
Admin Dashboard for Product and Order Management
Profile Update (User can edit info)
JWT-based secure authentication
Database updates directly via MongoDB.

Tech Used:-
Frontend: React, React Router, Axios, React Toastify, Vite
Backend: Node.js, Express, MongoDB, Mongoose, bcryptjs, JSON Web Token (JWT)
Other Tools: dotenv for environment variables, Cors
